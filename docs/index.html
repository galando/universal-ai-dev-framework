Commands

Essential commands for AI-assisted development

Core PIV commands that power your development workflow.

/piv_loop:prime

Prime Context

Load codebase context and understanding. Run at the start of each session or when switching contexts.

/piv_loop:plan-feature "description"

Create Implementation Plan

Create detailed implementation plan with deep codebase analysis. Use for complex features or refactoring.

/piv_loop:execute

Execute Plan

Execute implementation plan following strict TDD. Automatic validation runs after completion.

/validation:validate

Run Validation

Run full validation pipeline (automatic after execute). Includes environment check, compilation, tests, coverage.
